Return value

    forEach() → kuch return nahi karta (undefined deta hai).

    map() → ek naya array return karta hai jisme modified values hoti hain.

Use case

    forEach() → jab sirf array iterate karna hai aur side-effects (console.log, DB update, etc.) perform karne hain.

    map() → jab ek array se dusra transformed array banana hai.

Chaining

    forEach() → chaining possible nahi (kyunki return value undefined hai).

    map() → chaining possible hai (kyunki naya array return hota hai).

Performance

    Dono ki performance lagbhag same hai, but map() thoda efficient hota hai jab tumhe naya array hi banana ho.

    const arr = [1, 2, 3];

// forEach
arr.forEach(num => console.log(num * 2)); 
// Output: 2, 4, 6
// (kuch return nahi hota)

// map
const doubled = arr.map(num => num * 2);
console.log(doubled);
// Output: [2, 4, 6]
