2)  What’s the difference between inline, block, and inline-block elements? 

     Inline elements line ke andar hi rehte hain, matlab ye horizontally flow karte hain.
     Height & Width set nahi kar sakte directly.
     Examples: <span>, <a>, <strong>

     Block elements nayi line pe start hote hain aur full width le lete hain.
     Height & Width set kar sakte ho.
     Examples: <div>, <p>, <h1>

     nline-block elements inline ki tarah horizontally flow karte hain
     Lekin block ki tarah height & width set kar sakte hain
     Examples: <img>, <button>

3) How do you make a website mobile-friendly and accessible?     
    
  a) Use responsive layouts
   Use CSS Flexbox or Grid to create flexible layouts.
   Avoid fixed widths/heights; use %, vw, vh, or rem instead.

  b) Use media queries
  c) ARIA attributes
